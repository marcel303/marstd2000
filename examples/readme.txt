MARSTD/MARX EXAMPLES
--------------------

Use Dev-C++ to compile the supplied project files.

The examples may or may not require the following librairies:

	Allegro		A Game Programming Librairy.
	AllegroGL	OpenGL graphics drivers for Allegro.
	Mesa3D		Software OpenGL rendering. Only when compiling for DOS.	

All can be downloaded from SourceForge.

These examples should compile nicely under any platform supported by the above mentioned
librairies, provided you create your own make files.
The list of supported platforms includes amongst others: Win32, Linux/Unix and DOS.
